# 图标选择器组件说明

## 组件概述

`IconPicker` 是一个可视化的图标选择组件，替代了手动输入图标名称的方式。

**组件路径：** `web/src/components/IconPicker/index.vue`

## 功能特性

- ✅ 可视化图标选择界面
- ✅ 支持图标搜索功能
- ✅ 网格布局展示所有图标
- ✅ 实时预览选中的图标
- ✅ 支持清除选择
- ✅ 响应式设计

## 配置文件

**文件：** `web/src/config/icons.ts`

- 集中管理所有可用图标的映射关系
- 提供 `getIcon()` 函数将图标名称转换为组件
- 包含 100+ 个常用图标（基础、系统、监控、存储等分类）

## 使用示例

```vue
<template>
  <IconPicker v-model:value="currentIcon" />
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { IconPicker } from '@/components/IconPicker';

const currentIcon = ref('dashboard');
</script>
```
