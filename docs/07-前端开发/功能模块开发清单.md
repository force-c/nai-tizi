## 一、系统管理模块

### 1.1 用户管理

**页面路径：** `src/views/system/user/index.vue`

**功能清单：**
- [ ] 用户列表（分页、搜索、筛选）
- [ ] 新增用户
- [ ] 编辑用户
- [ ] 删除用户
- [ ] 批量删除用户
- [ ] 重置密码
- [ ] 用户详情查看
- [ ] 批量导入用户

**API接口：**
```typescript
POST /api/v1/user/page          // 分页查询
POST /api/v1/user               // 创建用户
GET  /api/v1/user/:id           // 获取详情
PUT  /api/v1/user/:id           // 更新用户
DELETE /api/v1/user/:id         // 删除用户
DELETE /api/v1/user/batch       // 批量删除
PUT  /api/v1/user/:id/password  // 重置密码
POST /api/v1/user/import        // 批量导入
```

**权限标识：**
- `user.read` - 查看用户
- `user.create` - 创建用户
- `user.update` - 更新用户
- `user.delete` - 删除用户
